# 📈 InvestPro

> Plataforma funcional de simulação financeira construída inteiramente com HTML5, CSS3 e JavaScript nativo (Vanilla JS).

---

## 💻 Sobre o Projeto

O **InvestPro** é uma aplicação web completa desenvolvida de forma 100% independente. O grande diferencial do projeto foi o desafio de encarar as regras de negócio, a lógica matemática financeira e o fluxo de dados diretamente no front-end na raça, sem depender de frameworks ou bibliotecas externas.

O simulador permite que o usuário realize um cadastro completo e receba projeções detalhadas de rentabilidade e evolução patrimonial em tempo real com base no seu perfil e no ativo selecionado.

---

## 🛠️ Tecnologias Utilizadas

A pilha de tecnologia foca na base sólida do desenvolvimento web:

*   **HTML5 Semântico:** Estruturação limpa do app com foco em acessibilidade.
*   **CSS3 Moderno:** Estilização avançada em *Dark Mode*, layouts responsivos e design de modais.
*   **JavaScript Nativo (Vanilla JS):** Manipulação dinâmica do DOM, validação de dados e lógica financeira.

---

## 🚀 Principais Funcionalidades Implementadas

*   **Sistema de Cadastro Completo:** Formulário estruturado com tratamento de inputs, máscaras de dados e validações em tempo real.
*   **Manipulação de Dados via Método GET:** Integração dinâmica de páginas (como a seção "Sobre") e controle de estados utilizando parâmetros e leitura diretamente na URL com JS nativo.
*   **Cálculo e Projeção Financeira:** Algoritmo em JavaScript que processa o aporte inicial, o prazo e a taxa do ativo para gerar o montante final estimado e o lucro líquido.
*   **Componentes Interativos:** Modais dinâmicos para visualização detalhada de ativos específicos (Fundos, Renda Fixa, CDBs) e telas limpas de conclusão de simulação.
*   **Efeitos Dinâmicos com Canvas:** Renderização em segundo plano de uma animação de partículas leve e performática programada via API de `canvas`.

---

## 📁 Estrutura de Arquivos Principal

O projeto foi organizado mantendo uma separação clara de responsabilidades:
*   `index.html` / `about.html` / `form.html` -> Estrutura das páginas e fluxos.
*   `styles/main.css` -> Identidade visual e layout completo.
*   `script/formAction.js` / `form.js` / `main.js` -> Toda a inteligência e lógica da aplicação.
