Sobre o Projeto

Conheça os fundamentos acadêmicos, os objetivos de engenharia de software e as tecnologias empregadas no desenvolvimento do InvestPro.

📚 Atividade Acadêmica

Este projeto foi desenvolvido como parte da Atividade Somativa 1 da disciplina Fundamentos de Programação Web, do curso de Análise e Desenvolvimento de Sistemas da PUCPR.

🚀 Competências Demonstradas

  • ✔Estruturação de páginas web responsivas.
  • ✔Organização de arquivos HTML, CSS e JavaScript.
  • ✔Validação de formulários no lado cliente.
  • ✔Manipulação de parâmetros da URL utilizando JavaScript.
  • ✔Criação de interface visual consistente entre múltiplas páginas.

💻 Tecnologias Utilizadas

HTML5 CSS3 JavaScript

Estruturação com HTML5, identidade visual com CSS3 e toda a interatividade — validações, máscaras, partículas e cálculos de projeção — com JavaScript puro.

🎯 Objetivo do Sistema

O InvestPro foi desenvolvido com o propósito de simular uma plataforma de investimentos, permitindo a navegação entre páginas, o preenchimento e validação de formulários, além da captura e exibição de informações enviadas pelo método GET.
